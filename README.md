# TeamPikachu Aerothon Project

## Overview

This project is a web application developed using React and Vite. It allows users to find the optimal flight path between two cities using weather APIs and OpenStreetMap. The application uses Dijkstra's algorithm to determine the shortest path, factoring in real-time weather conditions.

## Features

- Select departure and arrival cities
- View the optimal flight path on an interactive map
- Get real-time weather updates for the selected path
- User-friendly interface with city and date selection

## Technologies Used

- React
- Vite
- JavaScript
- OpenStreetMap
- OpenWeatherMap API
